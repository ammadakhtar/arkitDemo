# ARKit-Demo
![Platform](https://img.shields.io/badge/Platform-iOS-orange.svg)
![Languages](https://img.shields.io/badge/Language-Swift-orange.svg)

An ARKit demo application for displaying 3D models.

## Demo
- [Link](https://gph.is/g/Z7JW2GP)

## Features

- [x] Horizontal Plane Detection
- [x] Tap gesture to place 3D model into the ARScene.
- [x] Pan gesture to move the object.
- [x] Pich gesture to scale the object.
- [x] Rotation gesture to rotate the object.

## Requirements

- iOS 13.0+
- Xcode 11+
- Swift 5+
- Iphone 6s+

## Dependencies
None.

## Installation

Directly clone the repository and run the peoject using a physical device.
